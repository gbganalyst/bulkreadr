# bulkreadr: The Ultimate Tool for Reading Data in Bulk

Designed to simplify and streamline the process of reading and
processing large volumes of data in R, this package offers a collection
of functions tailored for bulk data operations. It enables users to
efficiently read multiple sheets from Microsoft Excel and Google Sheets
workbooks, as well as various CSV files from a directory. The data is
returned as organized data frames, facilitating further analysis and
manipulation. Ideal for handling extensive data sets or batch processing
tasks, bulkreadr empowers users to manage data in bulk effortlessly,
saving time and effort in data preparation workflows. Additionally, the
package seamlessly works with labelled data from SPSS and Stata.
